# H5 仿真程序说明

## 作品信息

标题：大学物理数值仿真实验平台

实验模块：

- 电磁学：带电粒子在电磁场中的运动仿真
- 物理光学：杨氏双缝、夫琅禾费单缝衍射与平面衍射光栅仿真

技术形式：HTML5 + CSS + JavaScript

运行环境：Edge、Chrome 等主流浏览器，直接打开 `index.html` 即可运行。

## 文件结构

- `index.html`：页面结构、实验切换、参数控件。
- `src/styles.css`：界面布局、响应式样式和可视化配色。
- `src/opticsModel.js`：物理光学强度模型和关键读数计算。
- `src/simulator.js`：电磁学 RK4 数值积分、Canvas 绘制和交互逻辑。
- `tests/`：夫琅禾费单缝衍射和平面衍射光栅模型的 Node 验证脚本。
- `assets/`：NuSEP 标识与素材副本。

## 核心功能

- 顶部可在“电磁学”和“物理光学”两个实验之间切换。
- 所有主要参数同时支持滑块调节和手动数值输入。
- 电磁学模块支持纯磁场螺旋、E×B 漂移、速度选择器、电场加速、负电荷偏转。
- 电磁学模块可拖动画布平移参考网格，并标注粒子起始位置。
- 物理光学模块支持双缝干涉、夫琅禾费单缝衍射、平面衍射光栅三个预设。
- 物理光学模块可调波长、屏距、屏幕跨度、缝距或光栅常数、单缝宽、光栅条数和相位差，其中缝距、光栅常数和单缝宽以微米为界面单位。
- 图表区在电磁学模式显示速率/动能曲线，在物理光学模式显示强度曲线和单缝包络。

## 操作建议

1. 选择“电磁学”，从“纯磁场螺旋”开始，观察磁场只改变速度方向、不改变动能。
2. 切换到“速度选择器”，调节 `vx0`，观察电场力和磁场力抵消的条件。
3. 选择“物理光学”，从“双缝干涉”开始，调节波长和缝间距，观察条纹间距变化。
4. 切换到“夫琅禾费单缝”，在几十微米量级调节单缝宽，观察中央亮纹宽度和暗纹位置变化。
5. 切换到“平面衍射光栅”，调节光栅常数和光栅条数，观察主极大位置和亮纹锐化。
